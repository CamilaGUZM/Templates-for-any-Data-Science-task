## Installation

To reproduce the analysis in this repository, follow the steps below.

### 1. Clone the repository

<!-- CHANGE: -->
```bash
git clone sshlink.git
cd repo name
```


### 2. Software requirements
<!-- CHANGE: -->
Ensure you have the following installed:
- Python 3.9 or higher
- pip (Python package manager)
Verify  Python installation:
python --version

### 3. Install dependencies

If a requirements.txt file is provided:

```bash
pip install -r requirements.txt
```
Otherwise, install the core dependencies manually:

<!-- CHANGE: -->
```bash
pip install numpy pandas matplotlib seaborn
```

### 4. Run the analysis
<!-- CHANGE: -->
Execute the scripts or notebooks included in the repository to generate the results.
If using Jupyter notebooks:

```bash
jupyter notebook
```

### 5. View the report
<!-- CHANGE: -->
The full rendered analysis is available in:

```text
Report.html
```
